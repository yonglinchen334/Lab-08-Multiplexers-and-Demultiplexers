## Lab Questions

### In plain English describe the function and use of a multiplexer.
Multiplexers allows the programmer to select one output out of multiple input signals via a selector switch. 
### In plain English describe the function and use of a demultiplexer.
Demultiplexers are like multiplexers, but does the opposite. It takes one input signal and broadcast it to multiple outputs via a selctor switch as well. 
### What other uses might these circuits have? (Think Shannon’s)
Multiplexers can be used to build any boolean logic function. They can be used to replace large gate networks, build programmable logic, create structured, hierachial logic, convert truth tables directly into hardware, and build larger multiplexers from smaller ones. Shannon's Expansion is important because it is a decomposed function that can represent multiple variables that are used as selector inputs which can then be used to build multiplexers.
